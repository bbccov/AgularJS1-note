使用 orderBy 过滤器 排序显示，可以使用 orderBy 过滤器

{{ x.Name }} {{ x.Country }}

使用 uppercase 过滤器 使用 uppercase 过滤器转换为大写:

{{ x.Name }} {{ x.Country | uppercase }}

显示序号 ($index) 表格显示序号可以在中添加 $index:

{{ $index + 1 }} {{ x.Name }} {{ x.Country }}

使用 $even 和 $odd

{{ x.Name }} {{ x.Name }} {{ x.Country }} {{ x.Country }}
